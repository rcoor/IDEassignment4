Show of hands

Principal Component Analysis of hands

We, the human race are quite alike. We have all heard the phrase that we are 99.9% the same – although that this is disputed it is clear that we carry a higher resembles between each other than for example to a Chimpanzee.

It is this fact that makes all our small differences even more interesting – we have a very clear basis of comparison. In this example we have 40 individual outlines of hands. Each comes as a set of x- and y-coordinates that can be visualized in a scatter-plot as points.

In the example below we can see that there is a clear difference between some of the shown hands. This might be due to different characteristics of the hands or way the hand was placed when it was drawn.

It's difficult for the human eye to extract any meaningful correlation from a plot as show above. A Principal Component Analysis (PCA) can be used in order to do a deeper analysis of features.

All hands on deck, a PCA plot

There are no clearly visible clusters among the 40 hands.

By selecting a point on the PCA we get the corresponding hand drawn on the other plot.
